# Connecting GitHub to Your Application \{#integrations_github-1}

Follow these steps to connect your GitHub account:

## 1. Access the Connector Page \{#integrations_github-2}

- Log in to your account
- Navigate to the Connector page
- Find the SCM Panel
- Click on the GitHub section

<img src="/docs/plexicus/connector.png" alt="Connector Page" />

## 2. Initiate the Connection \{#integrations_github-3}

- It will show tabs with the `Token` and `Installation ID`
- Select the `Token` tab
- Click the `Connect` button

<img src="/docs/plexicus/github/panel.png" alt="GitHub Panel" />

## 3. Authorize GitHub Access \{#integrations_github-4}

- Wait for the authorization process to complete
- Grant necessary permissions for Plexicus to access your GitHub account

<img src="/docs/plexicus/github/authorize.png" alt="GitHub Authorize" />

## 4. Install Plexicus Application \{#integrations_github-5}

- The Plexicus application will be installed on your GitHub account
- Select the repositories you want to use with Plexicus

<img src="/docs/plexicus/github/installation.png" alt="GitHub Installation" />
<img
	src="/docs/plexicus/github/install-authorize.png"
	alt="GitHub Authorize Installation"
/>

## 5. Finalize the Connection \{#integrations_github-6}

- You'll be redirected back to Plexicus
- Verify that the connector is successfully configured

## Next Steps \{#integrations_github-7}

You can now create an application using GitHub as your Source Control Management (SCM) system.

<img src="/docs/plexicus/applications.png" alt="Applications Page" />

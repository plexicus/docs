# Connecting Bitbucket Cloud to Your Application \{#integrations_bitbucket-1}

Follow these steps to connect your Bitbucket Cloud account:

## 1. Access the Connector Page \{#integrations_bitbucket-2}

- Log in to your account
- Navigate to the Connector page
- Find the SCM Panel
- Click on the Bitbucket Cloud section

<img src="/docs/plexicus/connector.png" alt="Connector Page" />

## 2. Initiate the Connection \{#integrations_bitbucket-3}

- It will show tab `Token`
- Click the `Connect` button

<img src="/docs/plexicus/bitbucket/panel.png" alt="Bitbucket Cloud Panel" />

## 3. Authorize Bitbucket Cloud Access \{#integrations_bitbucket-4}

- Wait for the authorization process to complete
- Grant necessary permissions for Plexicus to access your Bitbucket account

<img src="/docs/plexicus/bitbucket/authorize.png" alt="Bitbucket Authorize" />

## 4. Finalize the Connection \{#integrations_bitbucket-5}

- You'll be redirected back to Plexicus
- Verify that the connector is successfully configured

## Next Steps \{#integrations_bitbucket-6}

You can now create an application using GitHub as your Source Control Management (SCM) system.

<img src="/docs/plexicus/applications.png" alt="Applications Page" />

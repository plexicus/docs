# Organization & Account Settings \{#organization_client-1\}

This page is the operator's map of the Settings area. Each section below names the **UI label**, the **path** in the SaaS, and what you do there.

The Settings sidebar is reached from the user menu in the top-right corner of any page.

## Account \{#organization_client-3\}

Path: **Settings → Account** (`app.plexicus.ai/settings/account`).

Change your password (for accounts created with email + password — Google or SCM-OAuth users skip this section):

1. Enter your current password.
2. Enter the new password and confirm.
   - Minimum 8 characters
   - At least one uppercase letter
   - At least one number
   - At least one special character
3. Click **Save**.

Account deletion lives at **Settings → Account → Delete account** (`/settings/account/deletion`). Deletion is irreversible.

## Authentication (2FA) \{#organization_client-2\}

Path: **Settings → Authentication** (`app.plexicus.ai/settings/authentication`).

Enable two-factor authentication and manage authenticator app pairing. Full walkthrough: [Configure 2FA](/docs/recipes/configure-2fa).

## Plexalyzer Token \{#organization_client-9\}

Path: **Settings → Plexalyzer Token** (`app.plexicus.ai/settings/plexalyzer-token`).

Generate, view, and revoke the Plexalyzer service token used by self-hosted Plexalyzer workers to authenticate against the platform. Most SaaS users don't need this — it's intended for self-hosted scanning workers.

## Domain Verification \{#organization_client-10\}

Path: **Settings → Domain Verification** (`app.plexicus.ai/settings/domain-verification`).

Register and verify a custom domain to bind your tenant to. Full walkthrough: [Register a Domain](/docs/recipes/register-domain).

## Client (Organization profile) \{#organization_client-4\}

Path: **Settings → Client** (`app.plexicus.ai/settings/client`).

This is the organization-level profile page. Configure:

- **Organization name** — your company name as shown to your team.
- **Schedule scan** — pick a recurring scan cadence (None / Weekly / Monthly).
- **Advanced AI Analysis** — toggle deep-reasoning enrichment on findings.
- **Prioritization Managed by Plexicus** — let Plexicus recompute finding priority based on runtime context.

Click **Save** when done.

## CNAPP \{#organization_client-11\}

Path: **Settings → CNAPP** (`app.plexicus.ai/settings/cnapp`).

Cloud-native security configuration: which scanners run, severity thresholds, custom parameters per tool. See [Customize Scanners](/docs/recipes/customize-scanners) and [Configure Custom Parameters](/docs/recipes/configure-custom-parameters).

## Team \{#organization_client-8\}

Path: **Settings → Team** (`app.plexicus.ai/settings/team`).

Invite and manage team members:

1. Enter the new member's email.
2. Pick a role:
   - **Cyberoper** — security operator; triages findings, manages remediations.
   - **Developer** — engineer who fixes findings assigned to them.
   - (Admin invites are managed via the audit / superadmin path.)
3. Click **Invite**.

Role definitions: [Roles and Permissions](/docs/concepts/roles-and-permissions).

## Subscription \{#organization_client-12\}

Path: **Subscription Detail** (`app.plexicus.ai/payment`) — accessible from Settings or from the user-menu's "Subscription" link.

Sub-pages:

- **Information** (`/payment/info`) — current plan, payment method, billing address.
- **Invoices** (`/payment/invoices`) — download past invoices.
- **Cancel / Reactivate** (`/payment/cancel`).

For tier matrices and limits, see [Pricing Tiers](/docs/reference/pricing-tiers).

## Microsoft Marketplace \{#organization_client-13\}

If you onboarded via the Azure Marketplace, billing flows through your Azure subscription instead of Stripe. Activation status is at `app.plexicus.ai/microsoft-marketplace`.

## Language \{#organization_client-14\}

Path: **Settings → Language** (`app.plexicus.ai/settings/language`).

Switch the UI between English and Spanish.

## AI Provider \{#organization_client-15\}

OpenAI / Azure OpenAI / DeepSeek configuration is managed via the **CNAPP** section above (or via the API for self-hosted). See [Configure AI Provider](/docs/recipes/configure-ai-provider) for the full setup.

:::note Geo-region restrictions
On the SaaS, registration from some countries (notably Spain) is routed to a contact form. This is a regional regulation policy, not a per-organization setting. Self-hosted deployments don't apply geo-restrictions.
:::
